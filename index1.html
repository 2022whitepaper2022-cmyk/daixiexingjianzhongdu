核心问题：这个患者需要补碱吗？

面对这位96岁高龄、合并多器官功能不全（心、肾、贫血）、存在严重代谢性酸中毒（HCO₃⁻ 13.8 mmol/L） 但pH值尚在正常范围（7.35）的患者，我们面临一个复杂的临床决策： 是否应该给予碳酸氢钠补碱治疗？

"宁酸勿碱"：原则与理论基础

什么是"宁酸勿碱"

核心定义

"宁酸勿碱"是临床处理酸碱平衡紊乱的重要原则。其核心思想是， 在面对酸中毒时，除非达到非常严重的程度，否则应倾向于维持轻度的酸中毒状态， 而不是积极、大量地使用碱性药物来迅速纠正pH值。

强调：寻找并纠正根本原因，而非单纯追求pH值正常化

临床应用场景

  • 休克状态：组织低灌注导致的乳酸性酸中毒
  • 代谢性酸中毒：糖尿病酮症酸中毒、肾功能衰竭等
  • 脓毒症：pH > 7.15时无须补碱 [334]

为何要"宁酸勿碱"

生理代偿机制

波尔效应：酸中毒促进氧释放

当pH降低时，血红蛋白与氧亲和力下降，氧解离曲线右移，增加组织氧供

[356]

避免过度补碱风险

  • • 代谢性碱中毒：脑血管收缩、心律失常
  • • 容量负荷过重：加重心衰、肺水肿
  • • 电解质紊乱：低钾血症、低钙血症
  • • 氧释放障碍：可能加重组织缺氧

治疗核心

恢复组织灌注优于单纯纠正pH

酸中毒是组织缺氧的结果，是"标"而非"本"

液体复苏是纠正酸中毒的关键 [334]

不同指南补碱指征推荐阈值比较

指南/共识来源 pH阈值 HCO₃⁻阈值 备注

老年患者的特殊性

生理特点

  • • 生理储备差，代偿能力弱
  • • 对酸碱变化敏感性高
  • • 药物代谢能力下降
  • • 易发生电解质紊乱

临床考量

  • • 合并症多，需综合评估
  • • 心肾功能不全风险高
  • • 治疗耐受性低
  • • 个体化决策更重要

补碱的方法与策略

药物选择

5%碳酸氢钠

首选推荐
  • ✓ 作用迅速，直接起效
  • ✓ 疗效确切，经验丰富
  • ✗ 含钠量高，需注意容量
  • ✗ 产生CO₂，加重呼吸负担

乳酸钠

  • • 作用相对缓和
  • • 在肝脏代谢生成HCO₃⁻
  • ⚠ 肝功能不全禁用
  • ⚠ 乳酸性酸中毒禁用

氨基丁三醇(THAM)

  • • 不含钠离子
  • • 同时纠正代谢性和呼吸性酸中毒
  • ⚠ 可能引起低血糖、高钾
  • ⚠ 价格相对昂贵

经验性给药

小剂量起始

5%碳酸氢钠50-100ml缓慢静滴

  • • 适用于紧急情况
  • • 快速简便，安全性高
  • • 符合"宁酸勿碱"原则
  • • 便于后续调整

公式计算法

常用公式

5% NaHCO₃ (ml) = (24 - 实测HCO₃⁻) × 体重 × 0.3

通常给予计算剂量的1/2或2/3

  • • 提供相对精确的剂量参考
  • • 24为目标HCO₃⁻浓度
  • • 0.3为分布容积系数
  • • 仍需个体化调整

给药原则

少量分次，缓慢静滴

  • 避免一次性快速推注
  • 分次给药，间隔充分评估时间
  • 每次剂量1-2小时内滴完
  • 给机体充分适应时间

密切监测

必备监测项目
  • • 动脉血气分析（最重要）
  • • 电解质（特别是血钾、血钙）
  • • 生命体征（心率、血压、呼吸）
  • • 中心静脉压（CVP，如需要）

每次给药后1-2小时复查血气

病例讨论：如何决策？

患者是否符合补碱指征？

依据pH值：7.35（正常范围）

传统标准：未达到pH < 7.2的补碱阈值

但：pH正常完全依赖呼吸代偿（PaCO₂ 25.0）

代偿有限，一旦失代偿病情将急剧恶化

依据HCO₃⁻：13.8 mmol/L（严重降低）

低于传统阈值（HCO₃⁻ < 15）

符合KDIGO 2024指南（HCO₃⁻ < 16-18）

明确补碱指征，需积极干预

补碱的风险与获益权衡

潜在获益

  • 改善心肌功能

    纠正酸中毒可增强心肌收缩力，改善心输出量

  • 改善血管反应性

    提高血管对儿茶酚胺的反应性，稳定循环

  • 改善代谢状态

    为其他治疗措施创造有利条件

主要风险

  • 高龄因素

    96岁高龄，器官功能储备极差，耐受性低

  • 心衰风险

    proBNP 28700，容量负荷易诱发肺水肿

  • 肾功能障碍

    肌酐452，肾脏调节能力严重受损

个体化治疗方案建议

结论

谨慎补碱，小剂量起始

HCO₃⁻严重降低提示需干预，但风险极高，必须极谨慎

具体方案

5%碳酸氢钠 50ml（约30mmol）
• 1-2小时内缓慢静脉滴注
• 滴注后1-2小时复查血气
• 密切监测电解质和生命体征

核心要点

治疗原发病是关键
• 改善组织灌注
• 解除尿路梗阻
• 控制感染
• 纠正贫血

总结与展望

核心观点回顾

"宁酸勿碱"是重要原则，但非绝对

  • • 强调病因治疗的重要性
  • • 避免盲目过度补碱
  • • 在严重酸中毒时需灵活应用
  • • 需要综合评估能力

补碱需严格把握指征，个体化决策

  • • 不能仅依赖pH或HCO₃⁻阈值
  • • 全面评估患者整体状况
  • • 老年患者需更加谨慎
  • • 目标：安全水平而非完全正常

临床实践要点

重视病因治疗

  • • 补碱只是辅助手段
  • • 原发病治疗是根本
  • • 才能从源头阻断酸产生

临床决策总结

对于这位96岁高龄、合并严重心肾功能不全的患者，其代谢性酸中毒（HCO₃⁻ 13.8 mmol/L） 已达到多个指南推荐的补碱指征，尽管pH值尚在代偿范围。然而，补碱风险极高。

建议采取"谨慎补碱，小剂量起始"的策略

例如，可给予5%碳酸氢钠50ml缓慢静滴，并严密监测血气、电解质及生命体征。 治疗的核心在于积极处理原发病。
